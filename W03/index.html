Object Declaration

Objects are very similar to objects used in other OOP languages, they are much easier to use in JavaScript though becuase they can be definded without the need of createing a class first. The easiest and recommended way to declare an object is using the object literal notation 

const spiderman = {
    name: 'Spiderman',
    'real name': 'Peter Parker',
    height: 178,
    weight: 76,
    hero: true,
    villain: false,
    allies: ['IronMan','Captain America', 'The Falcon'],
    swing() {
        return 'Thwip, Thwip!';
    },
    catchPhrase: "Your friendly neighboorhood Spiderman!"
};

The parts that are in the object are called properties, and if a property is a function it is known as a method.

const superman = {
    fly() {
        return 'Up, up and away!'; 
    } 
};

Objects as Parameters

Objects can be used as parameteres for functions, this allows the arguments to be provided by and in any order.

function hello({name, catchPhrase}) {
    console.log(`Hi! My name is ${name} and I'm ${catchPhrase}`);
}
hello({catchPhrase: spiderman['catchPhrase'], name: spiderman['real name']});

JSON

JSON stands for JavaScript Object Notation, it is a very popular data-storage format. One thing that I think is awesome about JSON is that it can be used by not just JavaScript but a lot of other programming languages as well. 

const laptop = '{"brand": "Lenovo", "model": "Legion", '
    "series": "7i", "type": "Gaming", "specs": {"cpu": "Intel i7", 
    "gpu": "Nvidia RTX 2060"}}'

This object in it's current form is not yet a JSON, to convert it to a JSON you need to parse the string

JSON.parse(laptop);

You can also turn a JSON back into a string object by using the stringify() method

JSON.stringify(laptop);
