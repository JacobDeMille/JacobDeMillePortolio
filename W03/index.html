const spiderman = {

Objects can be used as parameteres for functions, this allows the arguments to be provided by and in any order.

JSON

JSON stands for JavaScript Object Notation, it is a very popular data-storage format. One thing that I think is awesome about JSON is that it can be used by not just JavaScript but a lot of other programming languages as well. 

const laptop = '{"brand": "Lenovo", "model": "Legion", '
    "series": "7i", "type": "Gaming", "specs": {"cpu": "Intel i7", 
    "gpu": "Nvidia RTX 2060"}}'

This object in it's current form is not yet a JSON, to convert it to a JSON you need to parse the string

JSON.parse(laptop);

You can also turn a JSON back into a string object by using the stringify() method

JSON.stringify(laptop);

Chapter 6: The DOM

The DOM stands for "The Document Object Model" and it allows you to access a web page and enable interations with the page. The DOM is a collection of nodes that form a tree-like structure. Everything on the wepage is treated as a node by the DOM. Everything in the document is a child of the root node, the HTML tag. The DOM is Language agnostic so it may be used in any programming language.

Getting Elelments

To get an element from the document you use the documnt key word. You can use this with diferent getters to get specific elements, such as getElementById(), getElementsByClassName(), getElementByTagName(), querySelector(), and querySelectorAll()

ID: Each Id selector should be unique to just one element. getElementById() is the most commonly used method of accessing elements on a web page.

Query Selectors: querySelector() method allows you to use CSS notation in order to find the first element that matches a CSS selector. querySelectorAll() will return a node list of all the elements that mat the CSS selector.
