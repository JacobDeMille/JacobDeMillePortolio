Chapter 8: Forms

A simple serch box and button using a form

There are many different events that come along with forms, one is the focus event and the blur event. Focus will be activated when the user focuses on the element. The blur event is the exact opposite, when the user moves away from the focus area the blur event is activiated.

Submitting a form

The submit button is used to submit forms. By using events you can do things like prevent the submission of a form before. This is helpful for validation purposes, you can prevent the form from submitting and then run some functions to check if all of the form fields have been filled out or not. Once everything has been properly filled out, you can then allow the form to be submitted.

New Attributes in HTML5

In HTML there were new attributes introduced for the input element.
Autofocus, gives focus to this element when a page loads.
Placeholder, will insert the value provided in the input field until the user enters some text
Maxlength, will limit the number of characters that can be entered in the field.
