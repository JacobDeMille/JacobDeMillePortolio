CSS3 Transforms and Transitions

The transform property lets you lets you translate, rotate, scale, and/or skew any element on the page. You should not declare a new transform element. This is because of the Cascade in CSS, with that a seconed transform will override the first one. If you want to have more than one transform you should you should provide a space-seperated list of transform functions

Translation functions allow you to move elements left, right, up, or down. These functions are similar to the behavior of position: relative. Transitions usually appear smoother because they require less cliet side processing then doing the same thing with JavaScript

Transition & Transform Example

All CSS used for the hover div
.hover {
    text-align: center;
    width: 100px;
    height: 25px;
    background: lightblue;
    transition-property: width, height, transform, font, color;
    transition-duration: 2s;
    transition-timing-function: ease-in-out;
  }

  .hover:focus {
    width: 300px;
    height: 100px;
    font-size: 60px;
    background: rgb(255, 89, 89);
    transform: rotate(360deg);
  }

The scale(x,y) function scales an element by the defined factors horizontally then vertically.

The rotate() function rotates an element around the point of origin by a specified angle value.

SVG stands for Scalable Vector Graphics, it is markup language that is similar to HTML and uses XML to allow you to describe vector graphics.

One of the major advantages of SVG is that you can use CSS to manipulate it. Because it uses XML to produce it you can't use all of the CSS that you would normally but it does have specific CSS rules that you can use to modify it.

Example of an SVG drawing

(Hover your mouse to see a fun transform/transitions)
// All mark up used to create the diamond
<svg style="position: absolute; width: 0; height: 0;" width="0" height="0" 
  version="1.1" xmlns="http://www.w3.org/2000/svg" 
  xmlns:xlink="http://www.w3.org/1999/xlink">
    <defs>
      <symbol id="icon-diamonds" viewBox="0 0 1024 1024">
        <title>diamonds</title>

    </defs>
</svg>
// Div that contains the diamond shape
<div class="diamond">
  <svg class="icon">
    <use xlink:href="#icon-diamonds"></use>
  </svg>
</div>
