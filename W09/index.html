The Document Object

The write() method simply writes a string of text to the page. If a page has already loaded, it will completely replace the current document. The use of document.write() is heavily frowned upon as it can only be realistically used by mixing JavaScript within an HTML document. There are still some extremely rare legitimate uses of it, but a ninja programmer will hardly ever need to use it.

HTML5 APIs

HTML5 is the latest version of the Hypertext Markup Language used to create web pages. HTML5 added a lot of new features to the specitication.

The specification for HTML5 contains quite a few APIs that help gain access to hardware like cameras, batteries, geolocation, etc.

One API that seems like it would be very useful when developing for mobile devices is Geolocation. This could help a developer a lot when building certain applications.
